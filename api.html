Quick Start

Get your first API response in under 2 minutes. Create an account, grab your API key, and make your first call.

1. Install the SDK

Bash 
# Python
pip install agroguard

# Node.js
npm install @agroguard/sdk

2. Make your first request

Python 
import agroguard

client = agroguard.Client(api_key="ag-your-api-key")

# Simple crop question
response = client.chat.create(
    model="agro-guard-v1",
    messages=[
        {"role": "user", "content": "What causes yellow leaves on maize?"}
    ]
)

print(response.choices[0].message.content)

# Disease diagnosis from image
diagnosis = client.vision.diagnose(
    model="agro-vision-v1",
    image="path/to/crop_photo.jpg",
    crop="maize",
    language="en"
)

print(diagnosis.disease)
print(diagnosis.confidence)
print(diagnosis.treatment)
cURL 
curl https://api.agroguardai.com/v1/chat/completions \
  -H "Authorization: Bearer ag-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "agro-guard-v1",
    "messages": [
      {
        "role": "user",
        "content": "What causes yellow leaves on maize?"
      }
    ]
  }'
Node.js 
import AgroGuard from '@agroguard/sdk';

const client = new AgroGuard({
  apiKey: 'ag-your-api-key'
});

const response = await client.chat.create({
  model: 'agro-guard-v1',
  messages: [
    { role: 'user', content: 'What causes yellow leaves on maize?' }
  ]
});

console.log(response.choices[0].message.content);

Authentication

All API requests require a valid API key sent in the Authorization header.

Header 
Authorization: Bearer ag-your-api-key
Security note: Never expose your API key in client-side code. Always make API calls from your server.

Base URL

URL 
https://api.agroguardai.com/v1

Models

AgroGuardAI offers three specialized models, each optimized for different agricultural tasks.

ModelDescriptionContextPrice (input / output)
agro-guard-v1 General agricultural knowledge and Q&A 128K tokens $0.50 / $1.50 per 1M tokens
agro-vision-v1 Image analysis: disease, pest, and nutrient ID 16 images + 32K $1.00 / $2.00 per 1M tokens
agro-field-v1 Field operations, mission planning, fleet control 64K tokens $2.00 / $4.00 per 1M tokens

Diagnose

Upload a crop image to receive disease/pest diagnosis with treatment recommendations.

POST /v1/vision/diagnose
Request Body 
{
  "model": "agro-vision-v1",
  "image": "base64_encoded_image_data",
  "crop": "maize",
  "language": "en",
  "include_treatment": true
}
Response 
{
  "id": "diag_abc123",
  "disease": "Northern Corn Leaf Blight",
  "confidence": 0.94,
  "severity": "moderate",
  "treatment": {
    "organic": ["Remove infected leaves", "Improve air circulation"],
    "chemical": ["Apply azoxystrobin-based fungicide"],
    "prevention": ["Rotate crops", "Use resistant varieties"]
  },
  "model": "agro-vision-v1",
  "usage": { "prompt_tokens": 1024, "completion_tokens": 256 }
}

Chat Completions

Conversational interface for agricultural queries. Compatible with OpenAI chat format.

POST /v1/chat/completions
Request 
{
  "model": "agro-guard-v1",
  "messages": [
    {"role": "system", "content": "You are an agricultural expert."},
    {"role": "user", "content": "Best time to plant cassava in Katsina?"}
  ],
  "language": "ha",
  "temperature": 0.7,
  "max_tokens": 1024
}

Vision

Analyze crop images for diseases, pests, nutrient deficiencies, and growth stage assessment.

POST /v1/vision/analyze

Accepts JPEG, PNG, and WebP images up to 20MB. Supports batch analysis of up to 16 images per request. Returns structured data including identified issues, confidence scores, affected regions, and recommended actions.

Field Operations

Schedule and manage autonomous hardware missions via API.

POST /v1/field/missions
Request 
{
  "model": "agro-field-v1",
  "mission_type": "scout",
  "field_id": "field_xyz789",
  "schedule": "2026-04-15T06:00:00Z",
  "devices": ["drone_001", "rover_002"]
}
